07 — Правила
Что обязательно
🎨
Только CSS-переменные
Никаких хардкодных цветов (#555, #888 и т.д.). Всегда используй переменные из :root.
✓ color: var(--mid)
✗ color: #666
🔤
Два шрифта — не больше
Barlow (sans) для интерфейса и кнопок. Libre Baskerville (serif) для заголовков и слоганов. Никакого Playfair Display или Inter.
✓ var(--font-serif)
✗ 'Playfair Display'
⚖️
Вес заголовков — 400
Все заголовки страниц на var(--font-serif) используют font-weight: 400. Жирность создаётся размером, не весом.
✓ font-weight: 400
✗ font-weight: 700 (serif)
🔘
Форма кнопок — эллипс
Все кнопки используют эллиптический border-radius, а не обычный круглый.
✓ border-radius: 81px / 39px
✗ border-radius: 50px
📐
Заголовки — clamp()
Размеры заголовков всегда fluid через clamp(). Стандарт для h1 страниц: clamp(2.5rem, 6vw, 5rem).
✓ clamp(2.5rem, 6vw, 5rem)
✗ font-size: 48px
🏷️
Лейблы секций — оранжевые
Все нумерованные лейблы секций (01 — Collection) используют цвет var(--orange), размер 0.65rem, letter-spacing 0.2em.
color: var(--orange)
font-size: 0.65rem
letter-spacing: 0.2em
🖼️
Фон сайта — только cream
Фон страниц всегда var(--cream). Белый (#fff) используется только для карточек и форм внутри страниц.
✓ background: var(--cream)
✗ background: white (страница)
📱
Брейкпоинты
Мобайл: max-width 600px (кнопки, padding). Планшет: max-width 768px (сетки, nav). Десктоп: от 769px.
@media (max-width: 600px)
@media (max-width: 768px)
✨
Чубзики — только оригинальные
6 декоративных элементов с mariachi.dscff.es: крокодил, русалка, курица, рука, мандала, звёздный цветок. Никаких AI-заменителей.
